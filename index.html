WEB

It is an information retrieval system that allows information on the Internet to be searched in hypertext and multimedia environments. It is a system that effectively retrieves various information on the Internet expressed in hypertext format and is the most widely used in the world.Hypertext is displayed in the form of reading pieces of information such as "documents" or web pages from a web server through a program called a web browser and outputting them on a computer monitor. The user can then follow the hyperlinks on each page to another document, or send a set of information to a server serving that page. The act of following a hyperlink is often referred to as web surfing or web browsing. And a set of web pages containing related contents is called a web site.
